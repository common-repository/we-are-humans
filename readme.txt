We are humans

== Description ==

We are people, not machines.

A little plug-in to show your humanity based on the humans.txt initiative.

For more information about the humans.txt initiative and some ideas how to populate the file, please visit the <a href="http://humanstxt.org/">humans.txt web site</a>.

== Installation ==

1. Upload the plugin files to the <code>/wp-content/plugins/we-are-humans</code> directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the "Plugins" screen in WordPress.
1. Use the Settings -> We are humans screen to configure the plugin.
1. Enter your We are humans contribution.
